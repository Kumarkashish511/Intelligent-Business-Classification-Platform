## 🚀 About The Project

**GovINTEL.AI** is an AI-powered web application that automatically converts business descriptions into accurate **National Industrial Classification (NIC) codes**.

Whether you're a government official, researcher, or business owner — just describe your business and the platform instantly maps it to the correct NIC code using cutting-edge AI.

---

## ✨ Key Features

- 🧠 **AI-Powered Classification** — Uses NLP & Semantic Search to map business descriptions to NIC codes
- 🌍 **Multilingual Support** — Accepts both speech and text input in multiple languages
- 📂 **CSV Batch Processing** — Upload bulk business data and get NIC codes for all at once
- 📜 **History Tracking** — Keeps a record of all past user queries
- 💾 **Real-time Database Storage** — All queries stored and retrievable instantly
- ⚡ **Fast & Accurate** — Optimized for speed with high classification accuracy

## ⚙️ How To Run

**Step 1 — Clone the repo:**
```bash
git clone https://github.com/Kumarkashish511/Intelligent-Business-Classification-Platform.git
```

**Step 2 — Install dependencies:**
```bash
pip install sentence-transformers pandas
```

**Step 3 — Run the app:**
```bash
python final.py
```

---

## 🎯 How It Works
```
User Input (Text/Speech)
        ↓
  NLP Processing
        ↓
Semantic Search on NIC Database
        ↓
  Top Matching NIC Codes
        ↓
  Stored in Database + Displayed to User
```
